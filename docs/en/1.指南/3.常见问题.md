# Frequently Asked Questions

## Drawing cannot be displayed


### 1. Version issue

Try to ensure that mxcad, mxdraw, and the [MxDraw Cloud Development Package](https://www.mxdraw.com/download.html) are the latest versions,

If the drawing cannot be displayed and there is no error reported in the browser console, it is usually a version issue.

Update to the latest version:

```bash
npm install mxcad@latest mxdraw@latest
```

### 2. Unsupported drawing format

If you can open the converted sample drawing that we provide normally, but your own converted drawing cannot display partly or at all.

Then you can convert several more drawings. If only this drawing has problems, it may be that the conversion program does not fully support the conversion of this drawing.

You can send this drawing to customer service so that we can fix this problem.
